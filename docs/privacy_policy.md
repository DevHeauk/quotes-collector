# 개인정보처리방침

**매일명언** (이하 "앱")은 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다.

## 1. 수집하는 개인정보

본 앱은 **개인정보를 수집하지 않습니다.**

- 회원가입이 필요하지 않습니다.
- 이름, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다.
- 사용자의 관심사 설정(상황, 키워드)은 기기 내 로컬 저장소에만 저장되며, 서버로 전송되지 않습니다.
- 좋아요한 명언 목록은 기기 내 로컬 저장소에만 저장됩니다.

## 2. 네트워크 통신

앱은 명언 데이터를 제공하기 위해 서버와 통신합니다. 이 과정에서:

- 사용자가 선택한 관심사 카테고리(상황, 키워드 그룹명)가 API 요청 파라미터로 전송됩니다.
- 이 데이터는 명언 추천 목적으로만 사용되며, 서버에 저장되지 않습니다.
- 사용자를 식별할 수 있는 정보는 전송되지 않습니다.

## 3. 제3자 제공

수집하는 개인정보가 없으므로 제3자에게 제공하는 정보도 없습니다.

## 4. 데이터 보관

- 로컬 데이터: 앱 삭제 시 함께 삭제됩니다.
- 서버 데이터: 사용자 관련 데이터를 서버에 저장하지 않습니다.

## 5. 아동 개인정보

본 앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다.

## 6. 변경 사항

본 방침이 변경될 경우 앱 업데이트 또는 공지를 통해 안내합니다.
