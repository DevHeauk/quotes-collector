# 명언 태깅 검증 기준 v1

캘리브레이션 40건에서 도출한 기준. 검증 에이전트가 참조하는 문서.

## 오분류 판정 기준

### 1. 확실한 오분류 (반드시 수정)

- **핵심 메시지와 무관한 키워드**: 명언이 말하는 메시지가 아닌, 저자의 직업/분야/표면 소재로 태깅한 경우
  - ❌ 에디슨(발명가) → "창의성" / 실제 메시지는 실패와 끈기
  - ❌ 교육자가 말한 인생 명언 → "교육" / 실제 메시지와 무관
  - ❌ 주시경 "말이 오르면 나라도 오르고" → "교육", "지식" / 실제 메시지는 공동체, 신념

- **상황 태그 톤 불일치**: 명언의 감정 톤과 상황이 맞지 않는 경우
  - ❌ 위로/회복형 명언 → "절망적일 때" (o: "힘든 시기를 보낼 때", "희망이 필요할 때")
  - ❌ 공직자의 책임 명언 → "관계의 소중함" (o: "자기 성찰")
  - ❌ 자기성찰 위트 → "일상의 소소함" (o: "자기 성찰")
  - ❌ 사상/비판 명언 → "용기가 필요할 때" (o: "새로운 관점이 필요할 때")

- **그룹명과 동일한 중복 키워드**: 같은 그룹의 구체적 키워드가 이미 있는데 그룹명도 함께 태깅
  - ❌ ["목표", "끈기", "동기부여"] → "동기부여"는 그룹명이며 목표/끈기와 중복
  - ✅ ["목표", "끈기", "인내"] → 각각 구체적 개념

- **번역문-원문 불일치**: 한국어 텍스트와 text_original이 서로 다른 명언인 경우 → 데이터 오류로 별도 보고

### 2. 보강 권장 (수정 우선순위 낮음)

- **키워드 부족 (1~2개)**: 명언의 핵심을 충분히 포착하지 못함. 3~5개 권장.
  - 단, 의미가 매우 단순한 명언은 2개도 허용
- **상황 부족 (1개)**: 2~3개가 적정. 다른 맥락에서도 쓸 수 있다면 추가.
  - 단, 매우 특정한 맥락의 명언은 1개도 허용

### 3. 허용 범위 (수정 불필요)

- 경계선 태그: 명확한 오분류는 아니지만 더 나은 태그가 있을 수 있는 경우 → 현행 유지
- 키워드 3~5개, 상황 1~3개 범위 내 → 큰 문제 없으면 통과
- 고사성어: 현대적 확장 해석이 아닌 본래 뜻 기준 분류 → 본래 뜻에 맞으면 통과

## 분류 원칙 (요약)

1. **핵심 메시지 기준** — 표면적 소재가 아닌 전달하려는 메시지로 분류
2. **키워드 3~5개, 상황 1~3개** 적정
3. **구체적 태그 선호** — "인생" 같은 포괄 태그보다 "자아", "의미" 등 구체적 태그
4. **상황은 사용자 입장** — "이 명언을 언제 보여줄 것인가"의 관점
5. **고사성어는 원래 맥락** 기준

## 상황별 태그 사용 가이드

| 상황 | 적합한 명언 톤 | 부적합 예시 |
|---|---|---|
| 절망적일 때 | 극한 상황 묘사, 바닥의 경험 | 가벼운 위로 |
| 힘든 시기를 보낼 때 | 위로, 공감, "괜찮다" 톤 | 강한 도전 메시지 |
| 희망이 필요할 때 | 밝은 전망, 빛이 온다는 메시지 | 현실 직시형 |
| 포기하고 싶을 때 | "끝까지", 끈기 독려 | 수용/놓아줌 |
| 자기 성찰 | 자신을 돌아보게 하는 통찰 | 타인 관계 중심 |
| 웃음이 필요할 때 | 유머, 위트, 재치 | 심각한 철학 |
| 일상의 소소함 | 가벼운 일상 관찰 | 자기성찰적 위트 |
| 새로운 관점이 필요할 때 | 패러다임 전환, 비판적 사고 | 단순 동기부여 |
